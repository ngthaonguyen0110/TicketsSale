# Tickets Sale
### Phần 1: Xác định vấn đề 
- Mục tiêu: Đánh giá hành vi của khách hàng khi sử dụng sản phẩm, từ đó có thể đưa ra các cải tiến mới
- Dữ liệu: Dữ liệu lịch sử đặt vé xem phim online của khách hàng giai đoạn 2019-2022
- Câu hỏi: Hành vi của khách hàng như thế nào, có những điểm gì đặc biệt; công ty có thể làm gì để tăng trưởng doanh số, cải thiện chất lượng sản phẩm
